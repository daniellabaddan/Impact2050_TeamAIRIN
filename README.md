## AIRIN

1. Clone the repository

   ```bash
    git clone https://github.com/daniellabaddan/Impact2050_TeamAIRIN.git
   ```

1. Install the dependencies

   ```bash
    npm install
   ```

1. Launch the app

   ```bash
    npm run start
   ```

1. Explore the app
   ```bash
    http://localhost:3000/
   ```
